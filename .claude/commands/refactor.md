# Refactor workflow

You are refactoring code in this repository. Follow `CLAUDE.md`.

## Required steps

1. Restate the refactor goal.
2. Identify behavior that must remain unchanged.
3. Check existing test coverage.
4. Add characterization tests if coverage is weak.
5. Refactor in small steps.
6. Run targeted tests after meaningful changes.
7. Run the repository validation commands from `CLAUDE.md`.
8. Summarize:
   - What structure changed.
   - Why it is better.
   - How behavior was preserved.
   - Commands run and results.

## Constraints

- Do not change public behavior unless explicitly requested.
- Do not combine refactors with unrelated formatting churn.
- Prefer readability and testability over clever abstractions.
